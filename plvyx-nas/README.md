# PlvyX NAS

## 1. 고객 요구사항

- 서비스 확장성을 고려하여 클라우드 기반 NAS 환경을 구축해야 한다.
- 전체 임직원 60명 중 약 10명이 파일 디렉터리 내에서 동시에 업무를 수행할 수 있어야 한다.
- Windows, macOS, Linux 등 다양한 운영체제와 디바이스 환경에서 안정적으로 접근할 수 있어야 한다.
