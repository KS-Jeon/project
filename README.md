# README.md

## 1. 레포지토리 소개

이 저장소는 업무를 진행하며 수행하는 다양한 프로젝트의 문서, 산출물, 참고 자료를 체계적으로 관리하기 위한 레포지토리입니다. 개별 프로젝트별로 자료를 정리하고, 설계 문서와 운영 관련 자산을 지속적으로 축적할 수 있도록 구성되어 있습니다.
