# README.md

## 1. 레포지토리 소개

업무 중 수행하는 클라우드 프로젝트를 Harness 엔지니어링 기법을 적용하여 관리하는 레포지토리입니다.

AI 에이전트(Claude / Codex / Gemini)와 Harness 엔지니어링 기법을 활용해 설계 → 구현 → 검증 흐름을 구현하고, Terraform / Kubernetes / Helm 기반 산출물을 프로젝트 단위로 일관되게 관리합니다.
